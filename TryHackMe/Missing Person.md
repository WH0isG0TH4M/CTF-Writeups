# Room: Missing Person

Difficulty: Easy

Objectives: Use your OSINT skills to help the police track down a missing person.

## Challenge Description
```
"My friend went on holiday in 2025 and shared some photos, but I haven’t heard from him since. Can you help me track him down for the police report?"
```
Download the zip file attached to this task and start your investigation.

Upon unzipping the file, I found two photos:

- food.jpg
- MotoGP.jpg

I viewed the photos using xdg-open file_name.

Using reverse image search on MotoGP.jpg, I found out that it was located in Indonesia, so I searched on Google for the full name of the circuit.

### Question 1: What is the commercial name of this circuit?

Answer: Pertamina Mandalika International Street Circuit

To find when the event happened, I searched:
https://racingcalendar.net/circuit/pertamina-mandalika-international-street-circuit/2025

I found that the event was held on 03–05/10/2025.

### Question 2: When did the event take place?

Answer: 03–05/10/2025

I also performed a reverse image search on food.jpg and found the name of the restaurant, “Cantina Mexicana.”

### Question 3: He told me he ate delicious Mexican food. What is the name of the restaurant?

Answer: Cantina Mexicana

Using exiftool, I also found when the photo was taken:
Date/Time Original: 2025:10:05 19:55:30

### Question 4: At what time was this photo taken?

Answer: 19:55:30
